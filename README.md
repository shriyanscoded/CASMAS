# üéì CasMas: Master Your Scientific Calculator

**CasMas** is a high-fidelity, educational simulator built to help students master the physical tools they use in exams. By replicating the exact layout and logic of the **Casio fx-991ES PLUS**, it provides a risk-free environment to practice complex syntax and function mapping.

## üöÄ Why CasMas?
Standard calculator apps prioritize efficiency, but students often struggle in exams because they aren't familiar with their **physical hardware**. CasMas focuses on:
* **Tactile Familiarity:** Matches button locations and secondary functions (`Shift`/`Alpha`) of the real-world device.
* **Educational Guidance:** Includes a specialized **Guide Mode** to teach students the "Why" and "How" behind complex functions.
* **Natural Display (V.P.A.M.):** Mimics the system for writing fractions and square roots exactly as they appear in textbooks.

## ‚ú® Features
